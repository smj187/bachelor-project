# Data

This object contains data that was loaded from the backend.

Properties:
Name Type Description
id Number

The id of the node.
label String

The label of the node. Available: string, null or empty string.
description String

The description of the node. Available: string, null or empty string.
type String

The type of the node. Available asset, control, risk requirement or custom.
attributes Object

An object containing a key-value-pair list.
tooltipText String

Some tooltip text that is shown while hovering a specific node.
parent Number | Array.<Number>

One parent id reference or multiple parent ids references.
children Array.<Number>

An array containing child node id references.
config Object

An object, which is contains key-value entries to override default node representations.
