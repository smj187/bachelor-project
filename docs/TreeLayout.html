TreeLayout

TreeLayout(customConfigopt, customEventsopt, customNodesopt)

This class depicts given data within a tree layout. The algorithm to achieve this visualization is based on the Reingold-Tilford Algorithm. The main calculation process is based on the initial work found in an article, but extended in such a way that it fits the needs for the defined scope of this project.
