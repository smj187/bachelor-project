ContextualLayoutConfiguration

ContextualLayoutConfiguration

This object contains default configuration for contextual layout representations.

Properties:
Name Type Default Description
layoutWidth Number 1200

The width used by the layout representation.
layoutHeight Number 800

The height used by the layout representation.
translateX Number -50

Adds additional X translation for all SVG elements before rendering.
translateY Number 0

Adds additional Y translation for all SVG elements before rendering.
animationSpeed Number 300

Determins how fast SVG elements animates inside the current layout.
hideOtherLayouts Boolean false

If set to true, other layouts are not visible.
spacing Number 32

Determins the minimal spacing between nodes.
renderingSize String min

Determins the node render representation. Available: "min" or "max".
assignedFocusDistance Number 800

Determins the distance between the assigned and focus node.
riskFocusDistance Number 500

Determins the distance between all risk nodes and focus node.
parentFocusDistance Number 80

Determins the distance between all parent nodes and focus node.
riskContainerNodeLimit Number 3

Limits how many nodes the risk container renderes.
riskContainerColumns Number 2

Limits how many columns the risk container has.
riskContainderBorderRadius Number 2

Determins the containers border radius.
riskContainerBorderStrokeColor String #888888cc

Determins the containers border color.
riskContainerBorderStrokeWidth Number 1.85

Determins the containers border width.
riskContainerBackgroundColor String #ff8e9e05

Determins the containers background color.
childContainerNodeLimit Number 6

Limits how many nodes the child container renderes.
childContainerColumns Number 3

Limits how many columns the child container has.
childContainderBorderRadius Number 5

Determins the containers border radius.
childContainerBorderStrokeColor String #888888cc

Determins the containers border color.
childContainerBorderStrokeWidth Number 1.85

Determins the containers border width.
childContainerBackgroundColor String #fff

Determins the containers background color.
parentContainerNodeLimit Number 6

Limits how many nodes the child container renderes.
parentContainerColumns Number 3

Limits how many columns the child container has.
parentContainderBorderRadius Number 5

Determins the containers border radius.
parentContainerBorderStrokeColor String #888888cc

Determins the containers border color.
parentContainerBorderStrokeWidth Number 1.85

Determins the containers border width.
parentContainerBackgroundColor String #fff

Determins the containers background color.
