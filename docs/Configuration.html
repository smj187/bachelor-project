Configuration

This object contains default configuration for grid layout representations.

Properties:
Name Type Default Description
limitColumns Number 3

Limits how many columns the layout has.
limitNodes Number null

Limits how many nodes are rendered.
translateX Number 0

Adds additional X translation for all SVG elements before rendering.
translateY Number 0

Adds additional Y translation for all SVG elements before rendering.
animationSpeed Number 300

Determins how fast SVG elements animates inside the current layout.
hideOtherLayouts Boolean false

If set to true, other layouts are not visible.
spacing Number 32

Determins the minimal spacing between nodes.
renderingSize String min

Determins the node render representation. Available: "min" or "max".
