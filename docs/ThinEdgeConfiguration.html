ThinEdgeConfiguration

This object contains default configuration for thin edge representations.

Sets the spacing used for padding between label and background.

Determins how fast SVG elements animates inside the current layout.

Determins the edge type. Available: "solid" or "dashed".

Determins the edges thickness.
